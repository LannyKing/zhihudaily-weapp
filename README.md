# zhihudaily-weapp
知乎日报微信小程序


微信小程序开发文档  
https://mp.weixin.qq.com/debug/wxadoc/dev/?t=20161107

1、下载微信小程序开发工具
2、微信小程序注册申请 ，获取AppID(手机预览需用到)
3、服务器配置 ，添加合法域名，每个月只可修改3次 ，可添加多个域名
4、下载开发工具后创建项目，输入申请的AppID，填写项目名称，导入该工程
5、项目预览体验
